# Case Study Directory

This directory contains work applying the MTG Knowledge Complex personality frame to real subjects. It is separate from the framework documentation (`docs/ARCHITECTURE.md`, `docs/PLAN.md`, etc.) because it is exploratory and empirical rather than normative.

## Purpose

The KC personality frame maps the 25-element simplicial complex (5 vertices, 10 edges, 10 faces) onto human personality. This directory develops and tests that application through concrete case studies, meta-analysis, and methodological feedback.

## Methodology Note

The observability tier model (documented in `feedback_observability.md`) is the key methodological contribution of this phase. It distinguishes:

- **Tier 1** — private/internal (requires subject self-report or guided introspection)
- **Tier 2** — low-discoverability, low-legibility public (requires prior context to locate)
- **Tier 3** — high-discoverability, high-legibility public (standard web-findable material)

Analyses that rely only on Tier 3 evidence are incomplete. Future case studies should deliberately incorporate Tier 1 and Tier 2 sources.
